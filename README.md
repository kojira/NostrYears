# NostrYears

2025年のNostr活動を振り返るWebアプリケーション。

## 機能

- **投稿数・文字数**: kind 1の投稿数と総文字数（URL除外）
- **長文記事**: kind 30023の投稿数と文字数
- **リアクション**: kind 6（リポスト）、kind 7（リアクション）、kind 42（チャット）の数
- **投稿画像数**: kind 1に含まれる画像URLの数
- **人気投稿**: 最もリアクションが多かった投稿
- **仲良しランキング**: インタラクション量に基づくTOP10
- **パーセンタイル表示**: 他ユーザーと比較した上位%表示

## 技術スタック

- React 18 + Vite + TypeScript
- Material UI (MUI) v5
- nostr-fetch / nostr-tools
- pnpm

## 集計期間

2025年1月1日 0:00:00 JST 〜 2025年12月1日 0:00:00 JST

## デフォルトリレー

- wss://r.kojira.io
- wss://yabu.me

## 開発

```bash
# 依存関係のインストール
pnpm install

# 開発サーバーの起動
pnpm dev

# ビルド
pnpm build

# プレビュー
pnpm preview
```

## 使用方法

1. npubまたは公開鍵（hex）を入力
2. 「集計開始」ボタンをクリック
3. 集計結果が表示される
4. NIP-07拡張機能がある場合、結果をリレーに投稿可能

## NIP-07対応

NIP-07対応のブラウザ拡張機能（nos2x、Albyなど）をインストールすると：

- ワンクリックで公開鍵を取得
- 集計結果をリレーに投稿
- 他ユーザーとの比較（パーセンタイル表示）
